## 이미지에서 도출된 핵심 원칙

### 원칙 1. Pool별 채권 구성 — 4가지 Import 양식
채권 데이터 Import 양식을 4개 유형으로 분리:
1. **A. 일반무담보** — 신용대출, 카드, 리스 등 무담보 연체채권 (35개 컬럼)
2. **B1. CCRS** — 채무조정채권 (신용회복위원회 채무조정채권) (163개 컬럼)
3. **B2. IRL** — 채무조정채권 (개인회생채권) (23개 컬럼)
4. **C. 담보** — 담보채권 (근저당, 신탁 등에 따라 담보 설정된 채권) (21개 컬럼)

→ 기존 단일 템플릿 → **4개 별도 템플릿**으로 변경 필요
→ `/template/` 폴더의 실제 엑셀 템플릿 참조하여 구현

### 원칙 2. 동일 Pool 내 복수 매도인 구조 (multi-seller)
- 1개 Pool = 복수 매도인 가능
- 매수인도 복수 가능

### 원칙 3. 열람 권한
- **매도인** — 본인의 거래 이력만 열람
- **매수인** — 본인이 등록한 Pool 이력만 열람 (해당 Pool의 매도인별 거래현황 정보 포함)
- **회계법인** — 모두 열람 가능

### 원칙 4. Pool 등록 절차
1. Pool 등록 시 **[거래 정보]란만** 회계법인 담당자가 기재
2. 해당 등록 화면에서 **채권 데이터 파일 Import**
3. Import된 채권 데이터 기준으로 **Pool별 현황, 매도인별 현황 필요 정보 자동 완성**
4. Import 양식 내 주요 수치는 통일된 명칭: 고객번호 / 채권번호 / 차주 구분 / 미상환채권잔액(OPB)
5. **양도횟수 ≥ 1** → 재매각 정보란에 추가 집계 (거래 전체 정보 + 재매각 정보 양쪽 포함)
6. 채권 데이터 지연 수령 시 **수기 기재 및 조정** 가능, 사후 Import 후 동기화 ([Sync] 버튼)

## 수정 항목

### MOD-01. 채권 Import 템플릿 4개 유형 분리 (CR-01 확장)
- **위치:** 관리자 → 채권 관리
- **현재:** 단일 템플릿 (14개 컬럼, 자체 정의 양식)
- **수정 내용:**
  1. 기존 단일 템플릿 폐기
  2. `/template/` 폴더의 4개 실제 엑셀 템플릿 기반으로 변경:
     - A_일반무담보 (35개 컬럼): 자산확정일~기타
     - B1_CCRS (163개 컬럼): 자산확정일~이자율재조정
     - B2_IRL (23개 컬럼): 자산확정일~사건번호
     - C_담보 (21개 컬럼): 자산확정일~미수이자
  3. Bond 모델 확장: 유형별 공통 필드 + 유형별 전용 필드
  4. Import 시 **채권유형(bond_type)** 선택 필수
  5. 템플릿 다운로드도 유형별 4개 제공
  6. **공통 핵심 필드**: 자산확정일, 금융회사명, 고객번호, 채권번호, 차주구분, 양도횟수, 미상환채권잔액(OPB)
- **난이도:** ★★★

### MOD-02. 채권 요약표 Matrix 형태 변경 (CR-02 관련)
- **위치:** 거래현황 상세 → 채권 정보 탭
- **현재:** 하단에 차주구분별/상품유형별/담보유형별/연체기간별 탭 집계 테이블
- **수정 내용:**
  1. 상단 요약 카드 유지, 구성 항목 수정: **총 차주수 / 총 채권수 / 총 OPB** (총 합계잔액 제거)
  2. 하단 기존 집계 테이블 삭제
  3. **Matrix 형태 채권요약표**로 대체 (엑셀 "채권 요약표" 시트 참조)
     - 4개 섹션: 일반무담보채권 / 채무조정채권(CCRS) / 채무조정채권(IRL) / 담보채권
     - 각 섹션 구성: 매도인(행) × 차주구분(개인/개인사업자/법인)(열) × 지표(차주수/채권수/OPB)
     - 합계 행/열 포함 (SUM 자동 계산)
  4. 데이터 소스: bonds 테이블에서 bond_type + creditor(금융회사명) + debtor_type 기준 집계
- **난이도:** ★★★

### MOD-12. 거래현황 "매도인별" 탭 추가 (이미지 신규)
- **위치:** 거래현황 페이지
- **현재:** Pool별 단일 목록
- **수정 내용:**
  1. **"Pool별" / "매도인별"** 2개 탭 추가
  2. **Pool별 탭** (기존): Pool 단위 정보 나열 (모든 매도인 포함)
  3. **매도인별 탭** (신규): 매도인 단위 정보 나열
     - 동일 Pool이라도 매도인별로 별도 행으로 표시
     - 각 행에 해당 매도인의 OPB, 매각가율만 표시
     - **매도인은 본인 정보만 열람** (동일 Pool 내 다른 매도인 정보 열람 금지)
     - 양수인 컬럼은 "—"로 마스킹
     - 회계법인은 전체 열람 가능
- **난이도:** ★★★
